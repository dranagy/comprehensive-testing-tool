expand_more Critical SQL Injection in Login Endpoint /api/v1/auth/login Active Injection

Description

The application is vulnerable to SQL injection via the 'username' parameter on the login form. An attacker can manipulate the SQL query to bypass authentication or extract data from the backend database.

Remediation

Implement parameterized queries or prepared statements for all database access. Do not concatenate user input directly into SQL strings.

Evidence
